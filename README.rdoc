= Rich-Support

A small package of libs providing your gem Rails 2 and 3 compatibility and some core class goodies

== Introduction

Rich-Support is a small module of E9s (http://github.com/archan937/e9s) which provides Rails 2 and 3 compatibility to your own created gem. It also adds a small amount of handy methods to some Ruby core classes.
